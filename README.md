 Low Power ROM using Clock Gating

 Overview
This project implements a low-power ROM using:
- Gray code addressing
- Clock gating (BUFGCE)
- Reduced switching activity

# Features
- 64x8 ROM
- Gray encoder for address optimization
- FPGA-optimized clock gating
- Reduced dynamic power

# Tools Used
- Verilog HDL
- Xilinx Vivado
